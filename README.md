# Mini SIEM Dashboard

Mini SIEM développé en Python avec FastAPI.

## Features

- Détection d’IP suspectes
- Détection brute force
- Détection horaires suspects
- Dashboard temps réel
- API FastAPI
- Parsing de logs
- Scoring d’alertes

## Stack

- Python
- FastAPI
- HTML/CSS/JavaScript
- JSON
